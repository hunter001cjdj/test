# Arena Duel Online

多人連線房間制槍戰射擊遊戲。玩家可以建立房間、分享房號給朋友、加入同一場對戰，在 5 分鐘內累積最多擊殺數的一方獲勝。

## 專案結構

- `index.html`：多人房間大廳、HUD、遊戲畫面
- `styles.css`：大廳與戰場樣式
- `app.js`：前端 WebSocket 連線、房間操作、玩家輸入、畫面渲染
- `server.js`：Node.js + WebSocket 房間伺服器與同步邏輯
- `package.json`：執行腳本與相依套件
- `GAME_PLAN.md`：多人版本開發 SOP 與進度紀錄

## 核心玩法

- 玩家可建立房間並取得 6 碼房號
- 另一位玩家輸入房號後即可加入同一個房間
- 每個房間最多 2 位玩家
- 房主可在玩家到齊後開始 5 分鐘對戰
- 使用 `WASD` 移動、滑鼠瞄準、左鍵或 `Space` 射擊
- 雙方被擊倒後會短暫重生
- 倒數結束後，以擊殺數較高者獲勝

## 執行方式

1. 安裝 Node.js 18+。
2. 在專案目錄執行：

```bash
npm install
npm start
```

3. 打開瀏覽器進入：

```text
http://localhost:3000
```

4. 開兩個瀏覽器視窗或兩台裝置：
   - 第一位玩家建立房間
   - 第二位玩家輸入房號加入
   - 房主按下開始對戰

## 目前功能

- 房間建立與加入
- 雙人房對戰同步
- WebSocket 即時位置、瞄準、射擊同步
- 房主開始對戰
- 5 分鐘倒數、擊殺計分、重生與結算
- HUD 顯示時間、血量、擊殺數與房間狀態

## 開發備註

- 目前採用 Node.js 原生 HTTP 伺服器搭配 `ws`
- 房間上限為 2 人
- 這一版是可運作的多人原型，後續可擴充更多地圖、武器、觀戰、排行榜與正式配對

## 後續可擴充方向

- 增加第三人以上玩家與觀戰模式
- 補上音效與命中特效
- 加入斷線重連與房間保留
