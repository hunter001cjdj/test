# Arena Duel

瀏覽器版單人槍戰射擊遊戲。玩家會在競技場中與 AI 對手對戰，5 分鐘倒數結束時，擊殺數較高的一方獲勝。

## 專案檔案

- `index.html`：遊戲頁面與 HUD 介面
- `styles.css`：畫面樣式、版面與視覺設計
- `app.js`：角色控制、敵人 AI、子彈判定、計分與倒數邏輯
- `GAME_PLAN.md`：開發 SOP 與進度追蹤清單

## 遊戲玩法

- 玩家使用 `W` `A` `S` `D` 移動
- 使用滑鼠移動準星
- 按滑鼠左鍵或 `Space` 射擊
- 玩家與敵人被擊倒後都會短暫延遲後重生
- 5 分鐘結束後，擊殺數較高者獲勝

## 如何執行

1. 直接用瀏覽器開啟 `index.html`
2. 按下 `開始遊戲`
3. 開始對戰

## 目前功能

- 5 分鐘倒數計時
- 玩家移動與射擊
- 敵人 AI 移動、追擊與開火
- 子彈碰撞與傷害判定
- 死亡、重生與擊殺計分
- 勝負結算畫面
- 重新開始功能
